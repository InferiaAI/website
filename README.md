# Inferia Landing & Documentation

This repository contains the landing page and documentation for InferiaLLM.

## Project Structure

This repository contains the source code for the Inferia website and documentation.


## Getting Started

To run the website locally:

1. Navigate to the `docs/` or `landing` directory:
   ```bash
   cd docs or cd landing
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
