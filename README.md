# Inferia Landing & Documentation

This repository contains the landing page and documentation for InferiaLLM.

## Project Structure

- `landing/`: The main landing page built with React, Vite, and Tailwind CSS.
- `docs/`: Product documentation powered by Mintlify.

## Getting Started

### Landing Page

To run the landing page locally:

1. Navigate to the `landing/` directory:

   ```bash
   cd landing
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

### Documentation

The documentation is located in the `docs/` directory. It uses Mintlify for rendering.

To preview documentation locally (requires [Mintlify CLI](https://docs.mintlify.com/get-started/deployment)):

```bash
cd docs
mintlify dev
```
