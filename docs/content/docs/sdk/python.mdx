## Installation

```bash
pip install inferia
```



## CLI Reference

The `inferia` package comes with a built-in CLI for managing the entire platform.

```bash
usage: inferia [-h] {init,filtration-gateway,inference-gateway,orchestration-gateway,api-start} ...
```

### Commands

#### `inferiallm init`
Initializes the database schema.
- **Usage**: `inferiallm init`
- **Requires**: valid `POSTGRES_DSN` in `.env`.

#### `inferiallm api-start`
Starts the full stack (Inference, Filtration, Orchestrator + Worker) in a single process.
- **Usage**: `inferiallm api-start`
- **Options**:
  - `--host`: Bind host (default: `0.0.0.0`)
  - `--port`: Bind port (default: `8000`)
  - `--workers`: Number of worker processes.

#### `inferiallm inference-gateway`
Starts only the Inference Gateway.
- **Port**: Defaults to `8001`.

#### `inferiallm filtration-gateway`
Starts only the Filtration Gateway.
- **Port**: Defaults to `8000`.

#### `inferiallm orchestration-gateway`
Starts only the Orchestration Gateway (Control Plane).
- **Port**: Defaults to `8080`.
